This app is a portfolio project that I did to show my skills as a web developer. Its main purpose is to display basic information about countries, such as their capitals, population, etc. Simply type in a country name or its ISO A2 or ISO A3 country code to see the results - map should be centered and highlighted around your selection, and a dismissable info window should show up.
